COLLECTION POSTMAN DA API DA AUTOMATIONEDGE
Com esse Collection do aplicativo PostMan você consegue entender como utilizar as API da AutomationEdge dentro do seu processo.

Sobre este fluxo de automação:
Com esse Collection do aplicativo PostMan você consegue entender como utilizar as API da AutomationEdge dentro do seu processo. Normalmente o ciclo deve ser:

Autenticação – Geração de Token
Requisição – Execução de um Fluxo
Logout – Fechar a sessão.
Assim você fica com seu ambiente seguro.
